# genai3


Make an env and use the reuirements.txt to get the prerequisite packages
If you do this then you can simply run the cells without doing any of the package installs
just do the imports and then everything below that and ignore the install cells
pip install -r requirements.txt


Alternatively, use google colab and be on a gpu session. The installs you need for colab are in a labelled cell at the top.
Model outputs are just straight written to the directory where the repo is, the model scores are not written to disk at all and stay in the notebook cell runtime results
